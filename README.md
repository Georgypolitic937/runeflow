## 🛠️ Troubleshooting

### The app does not open

- Try running it again as a normal user
- Check that Windows did not block the file
- Move the file to a simple folder like Desktop
- Make sure the download finished fully

### Windows says the file is unsafe

- Check that you downloaded it from the official repository link
- Right-click the file and look for unblock options in file properties
- Try downloading it again from the same page

### The app cannot see my screen

- Make sure screen access is allowed
- Close remote desktop or screen-sharing tools if they interfere
- Use one monitor if the app has trouble with multiple screens

### The app clicks the wrong place

- Make your screen zoom and display scaling simpler
- Close pop-ups before starting
- Give shorter instructions
- Move the window you want the app to use to the front

### The AI does not respond

- Check your internet connection
- Confirm your model key or login details
- Try another supported model if the app allows it
- Restart the app and try again
